# Healthcare Data Warehouse ETL Pipeline

## Dataset

This project uses the Healthcare Dataset from Kaggle.
Due to dataset size and licensing considerations, the raw file is not included in
this repository.

Download the dataset:  
[Kaggle: Healthcare Dataset](https://www.kaggle.com/datasets/prasad22/healthcare-dataset?resource=download)

Add the dataset file to:  
`data/healthcare_dataset.csv`
